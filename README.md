# Stroop-Test
For use in sport science research

## Functionality

This is an interactive Stroop Test application designed for sport science research. The test presents color words displayed in mismatched colors, requiring participants to identify the actual color of the text (not the word meaning) under time pressure.

### Key Features

- **30-minute test duration** with continuous color word presentation
- **Color responses**: Users respond by pressing keys (R=Red, B=Blue, G=Green, L=Purple, O=Orange)
- **Timed responses**: Each word is displayed for 2 seconds
- **Strategic time windows**: Performance is tracked in three specific intervals:
  - 0-2 minutes (initial response)
  - 14-16 minutes (mid-test)
  - 28-30 minutes (fatigue assessment)
- **Real-time feedback**: Immediate visual feedback for correct/incorrect responses
- **Data logging**: Accuracy and response data collected for analysis

### How to Use

1. Open `src/main.html` in a web browser
2. Click "Start Test" to begin the 30-minute assessment
3. For each color word shown, press the key corresponding to the **color of the text**, not the word itself
4. Results are displayed on-screen and logged for research purposes
